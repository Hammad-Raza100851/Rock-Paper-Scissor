# 🪨✂️📄 Rock-Paper-Scissors Game

A fun and simple **Rock-Paper-Scissors** game in **Python**! Play against the computer and see who wins. 🏆

---

## 🎮 Features

- Choose between **Rock (r)** 🪨, **Paper (p)** 📄, or **Scissors (s)** ✂️  
- Computer makes a **random choice** 💻  
- **Scores are tracked** for both you and the computer  
- Announces the **winner** at the end of the game 🎉  
- Continue playing until you decide to **exit (e)** ❌
